# Media Convert

`media-convert` is a terminal media conversion tool backed by `ffmpeg` and `ffprobe`.

Location:

- app: `/Users/rafaelbm/Dev/AI/media-convert-tool/app.py`
- launcher: `/Users/rafaelbm/.local/bin/media-convert`

Run:

```bash
media-convert
```

Help:

```bash
media-convert --help
```
